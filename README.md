# Perennial React Router Frontend

Minimal health-check UI that calls the Go API `/health` endpoint.

## Run locally
- Install deps: `npm install`
- Start dev server: `npm run dev` (http://localhost:5173)
- Ensure the Go API is running on port 8080; the page loads `/health` on render.

## Configuration
- Override backend URL with env: `API_BASE_URL=https://api.example.com npm run dev`

## What it shows
- Green status when `/health` returns `ok`
- Server time from the response
- Error details if the request fails
